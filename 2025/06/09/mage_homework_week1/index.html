VMware 17 下载安装及虚拟机系统配置教程

第一部分：VMware Workstation 17 下载与安装

1. 下载 VMware Workstation 17

  1. 访问 VMware 官方网站：https://www.vmware.com
  2. 导航至"产品" > “Workstation Pro”
  3. 选择"下载试用版"或购买正式版
  4. 根据您的操作系统选择 Windows 或 Linux 版本

2. 安装 VMware Workstation 17

  1. 双击下载的安装程序（如：VMware-workstation-full-17.x.x-xxxxxx.exe）
  2. 接受许可协议
  3. 选择安装位置（建议保持默认）
  4. 勾选"增强型键盘驱动程序"（可选）
  5. 点击"安装"按钮
  6. 安装完成后点击"完成"
  7. 首次运行时需要输入许可证密钥（试用版有30天试用期）

第二部分：在 VMware 中安装 Rocky Linux 9.4

1. 准备 Rocky Linux 9.4 ISO 镜像

  1. 访问 Rocky Linux 官网：https://rockylinux.org
  2. 下载 Rocky Linux 9.4 ISO 镜像（选择适合的版本，如 Minimal ISO）

2. 创建 Rocky Linux 虚拟机

  1. 打开 VMware Workstation
  2. 点击"创建新的虚拟机"
  3. 选择"典型(推荐)“，点击"下一步”
  4. 选择"安装程序光盘映像文件(iso)"，浏览选择下载的 Rocky Linux ISO
  5. 设置虚拟机名称和位置
  6. 指定磁盘容量（200GB），选择"将虚拟磁盘拆分成多个文件"
  7. 点击"完成"开始创建虚拟机

3. 安装 Rocky Linux 9.4

  1. 启动虚拟机，进入安装界面
  2. 选择"Install Rocky Linux 9"
  3. 选择语言（如 English）
  4. 配置安装目标（磁盘分区，建议使用自动分区）
  5. 设置网络和主机名（建议启用网络）
  6. 设置 root 密码和创建用户
  7. 点击"Begin Installation"开始安装
  8. 安装完成后重启

第三部分：在 VMware 中安装 Ubuntu 24.04.1 桌面版

1. 准备 Ubuntu 24.04.1 ISO 镜像

  1. 访问 Ubuntu 官网：https://ubuntu.com/download/desktop
  2. 下载 Ubuntu 24.04.1 LTS 桌面版 ISO 镜像

2. 创建 Ubuntu 虚拟机

  1. 在 VMware 中点击"创建新的虚拟机"
  2. 选择"典型(推荐)“，点击"下一步”
  3. 选择"安装程序光盘映像文件(iso)"，浏览选择下载的 Ubuntu ISO
  4. 设置虚拟机名称和位置

  7. 创建用户账户和密码
  8. 点击"Install Now"开始安装
  9. 安装完成后重启

2. 配置网络（桥接模式或 NAT 端口转发）

  1. 关闭 Ubuntu 虚拟机

  4. 添加端口转发规则（如：主机端口2222转发到虚拟机22端口）

第五部分：使用 MobaXterm 远程登录 Ubuntu

1. 下载安装 MobaXterm

  1. 访问 MobaXterm 官网：https://mobaxterm.mobatek.net
  2. 下载 Portable 或 Installer 版本
  3. 运行安装程序或解压便携版

2. 配置 SSH 连接

  1. 打开 MobaXterm
  2. 点击"Session" > “SSH”
  3. 输入 Ubuntu 的 IP 地址（桥接模式直接使用虚拟机IP，NAT模式使用主机IP）
    • 如果是 NAT 端口转发，端口填写2222（或你设置的其他端口）
  4. 输入 Ubuntu 用户名
  5. 点击"OK"保存会话

3. 首次连接

  1. 双击保存的会话
  2. 首次连接会提示接受主机密钥，点击"Yes"
  3. 输入用户密码
  4. 成功登录后即可使用终端操作 Ubuntu 系统

4. 高级功能

  1. 文件传输：MobaXterm 左侧有文件浏览器，支持拖放上传下载
  2. X11 转发：默认启用，可运行图形程序
  3. 多标签管理：可以同时打开多个 SSH 会话
  4. 保存密码（可选）：在会话设置中可以保存密码（安全性较低）

常见问题解决

  1. 无法连接 SSH：

    • 检查 Ubuntu 防火墙：sudo ufw status
    • 临时禁用防火墙：sudo ufw disable
    • 或允许 SSH：sudo ufw allow ssh

  2. IP 地址变化：

    • 考虑在路由器中为虚拟机 MAC 地址分配静态 IP
    • 或在 Ubuntu 中配置静态 IP

  3. 性能优化：

    • 在 VMware 设置中分配更多 CPU 和内存资源
    • 安装 VMware Tools 增强性能

Shell 总结：概念与常见类型

一、什么是 Shell？

Shell 是操作系统提供的用户界面，充当用户与操作系统内核（Kernel）之间的桥梁。它主要有两大功能：

  1. 命令解释器：接收用户输入的命令，解释并传递给内核执行
  2. 脚本环境：支持编写自动化脚本程序（Shell Script）

Shell 的核心作用：

  • 执行命令和程序
  • 管理文件和目录
  • 控制进程
  • 环境变量管理
  • 编程和自动化

二、Shell 的分类

1. 按交互方式分类

  • 命令行 Shell（CLI）：如 Bash、Zsh
  • 图形 Shell（GUI）：如 Windows Explorer、GNOME Shell

2. 按功能特性分类

  • Bourne 系（sh 风格）：sh、bash、ksh、zsh
  • C 系（csh 风格）：csh、tcsh

三、常见 Shell 类型

Shell 名称 全称/含义 特点 常见系统
sh Bourne Shell 最古老的 Unix shell，所有 shell 的基础 传统 Unix
bash Bourne-Again Shell sh 的增强版，Linux 默认 shell Linux, macOS (2019前)
zsh Z Shell 强大的交互功能，支持插件和主题 macOS (10.15+默认)
ksh Korn Shell 结合了 sh 和 csh 的优点 AIX, HP-UX
csh C Shell 语法类似 C 语言 BSD 系统
tcsh TENEX C Shell csh 的增强版 某些 BSD 系统
fish Friendly Interactive Shell 用户友好，自带语法高亮 跨平台
dash Debian Almquist Shell 轻量级，速度快，适合脚本 Debian/Ubuntu (脚本默认)

四、主流 Shell 对比

  1. bash (最流行)

    • 优点：功能全面，兼容性好，文档丰富
    • 缺点：配置复杂，启动速度较慢

  2. zsh (现代选择)

    • 优点：强大的自动补全，主题支持，兼容 bash
    • 缺点：配置复杂（通常用 oh-my-zsh 管理）

  3. fish (新手友好)

    • 优点：开箱即用，自动建议，彩色显示
    • 缺点：语法不兼容 bash

  4. dash (高效脚本)

    • 优点：执行速度快，资源占用少
    • 缺点：交互功能弱

Linux 常用基本命令用法总结（结合 man 命令）

一、文件与目录操作

1. ls - 列出目录内容

常用选项：

  • -l 长格式显示（权限、大小、时间）
  • -a 显示隐藏文件（以.开头）
  • -h 人类可读的文件大小（如 KB/MB）
  • -t 按修改时间排序
  • -r 反向排序

2. cd - 切换目录

特殊路径：

  • cd ~ 返回家目录
  • cd - 返回上一个目录
  • cd .. 返回上级目录

3. pwd - 显示当前目录

4. mkdir - 创建目录

常用选项：

  • -p 递归创建多级目录
  • -m 设置权限（如 -m 755）

5. rm - 删除文件/目录

危险选项：

  • -r 递归删除（用于目录）
  • -f 强制删除（不提示）
  • -i 交互式删除（推荐新手使用）

二、文件操作

1. cp - 复制文件

常用选项：

  • -r 递归复制目录
  • -i 覆盖前提示
  • -v 显示复制过程
  • -p 保留文件属性

2. mv - 移动/重命名文件

注意：

  • 可用于重命名：mv oldname newname
  • -i 覆盖前提示

3. cat - 查看文件内容

常用组合：

  • cat file1 file2 > combined 合并文件
  • cat -n file 显示行号

4. less/more - 分页查看文件

操作键：

  • 空格：下一页
  • b：上一页
  • /pattern：搜索
  • q：退出

5. head/tail - 查看文件开头/结尾

三、权限管理

1. chmod - 修改权限

两种模式：

  • 数字模式：chmod 755 file（rwxr-xr-x）
  • 符号模式：chmod u+x file（给所有者增加执行权限）

2. chown - 修改所有者

常用：

  • chown user file
  • chown :group file
  • -R 递归修改

四、系统信息

1. ps - 查看进程

常用组合：

  • ps aux 查看所有进程
  • ps -ef 完整格式列表
  • ps -u username 查看指定用户进程

2. top/htop - 动态进程查看

操作键：

  • q：退出
  • M：按内存排序
  • P：按CPU排序
  • k：杀死进程

3. df - 磁盘空间

4. du - 目录大小

五、网络相关

1. ping - 测试网络连接

常用：

  • -c 4 发送4个包后停止
  • -i 2 每2秒发送一次

2. ifconfig/ip - 网络接口

3. netstat/ss - 网络连接

六、搜索与查找

1. find - 文件查找

2. grep - 文本搜索

常用选项：

  • -i 忽略大小写
  • -r 递归搜索
  • -n 显示行号
  • -v 反向匹配

七、压缩解压

1. tar - 打包/解包

选项说明：

  • -c 创建
  • -x 解压
  • -z gzip压缩
  • -v 显示过程
  • -f 指定文件名
